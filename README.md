## About This Project

This is a **Product Marketing Manager (PMM) take-home assignment** for RevenueCat: an updated **"For Product" landing page** targeting Product Managers, refreshing the messaging and feature narrative on the current page.

The work is judged on:
- **Narrative & positioning quality** — does the page tell a cohesive story about RevenueCat's value for PMs?
- **Visual storytelling** — does the design reinforce the message?
- **AI fluency** — how well were AI tools integrated into the process?

## What's Included

- **Research & findings** (`docs/`) — audience personas, competitive analysis, product context, current page teardown, proof/credibility
- **Positioning & messaging brief** (`docs/05-positioning-and-narrative.md`) — core pillars, value props, section arc
- **Design tokens & brand guidelines** (`design.md`) — visual system measured from the live page
- **Live site** (`site/`) — responsive HTML/CSS built with Tailwind, deployed on Vercel
- **Process documentation** (`docs/00-process-and-assumptions.md`) — research methods, tooling used, assumptions logged
